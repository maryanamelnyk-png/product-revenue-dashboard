## Revenue Analysis Dashboard

### Project Overview
This project focuses on analyzing company revenue to understand trends, key drivers, and user monetization patterns.

### Objectives
- Analyze revenue dynamics over time  
- Identify key products and regions  
- Calculate core metrics: Paid Users and ARPPU  
- Explore transaction distribution  

### Tools
- Tableau  

### Key Insights
- Revenue shows significant fluctuations: after a peak (~150K), there is a sharp drop followed by recovery → possible seasonality or product changes  
- USA is the main revenue driver  
- Main App and Customer Success contribute the most revenue  
- Paid Users and ARPPU grow until mid-year, then decline  

### Recommendations
- Investigate reasons for the post-peak drop  
- Scale successful strategies from USA  
- Analyze pricing and user behavior changes
